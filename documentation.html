Introduction

The goal of Damus Engine is to create games in pygame, a lot easier. The game is organized in scenes, there are a lot of useful functions, physics, collisions, text, etcetera it's already created and ready to implement. Everything is customizable. The engine take inspiration by the way the Unity Engine does things.

Quick Start

To start, import everything from the engine:

from damusengine import *
then create an instance of the game class
game = Game()
if you want to do something custom when quitting,create an onQuit function. Remember to add game.Quit() or Quit() in it or you won't be able to close the game.
def OnQuit(): #some cool stuff here game.Quit()
then setup the game object. No argument is needed, put them if you want to change some window properties. In this example i put them all
game.Setup((800,800),60,OnQuit,"green")
to start the game you need at least one scene, so let's create one. All the argument are not needed, but recommended. If you don't pass them you'll have to add the scene manually. If you want, create an update function to pass to the contructor, that will be called every frame. Delta time is needed as an argument
def Update(dt): print("this is the update function!") scene = Scene(game,True, Update)
now let's create one test object
testObj = GameObject(scene,EmptyImage((100,100),"green"),(400,400))
finally you can start the game. Check the rest of the documentation for more.
game.Start()

Game Class

Constructor
Instantiate the class without parameters.
game = Game()

Attributes
  • clock: used in the update. should not be used (pygame.time.Clock)
  • desiredFPS: fps you want to make (int)
  • sizes: window sizes. Cannot be changed (tuple)
  • title: window title (str)
  • onQuitFunction: function called on quit (function)
  • screen: window surface. Should not be changed (pygame.Surface)
  • bgColor: window background fill color (str or tuple)
  • gravityConstant: gravity constant used in rigid bodies (float)
  • scenes: dictionary (with int keys) that stores scenes (dict)
  • activeScene: the active scene. Should not be changed maually (Scene)
  • dt: delta time. Changing it is useless (float)
  • lastTime: used for dt calculation, tho it's useless to use (float)
  • hasSetup: makes sure you call Setup(). Cannot be changed (bool)
Functions
Setup() -> None

  • Parameters

  • sizes: the window sizes (tuple) (not required)
  • desiredFPS: the fps you want (int) (not required)
  • onQuit: the on quit function (function) (not required)
  • bgColor: window background color (str or tuple) (not required)

Setup the attributes and the game window. It's needed to start the game.

Start() -> None

No parameters needed

Start the game. If the Setup() hasn't been called or if there is no active scene it will raise errors.

Quit() -> None

No parameters needed

Quit the game.

Update() -> None

Should never be called manually.

AddScene() -> None

  • Parameters

  • scene: the scene to add (Scene)
  • isActive: if the scene is the new active (bool)

Add a scene to the editor. Can automatically set it to active if the second parameter is True

LoadScene() -> Scene

  • Parameter

  • sceneIndex: the index of the scene (int)

Set the scene with that index as active. Also returns that scene.

RemoveScene() -> None

  • Parameter

  • sceneIndex: the index of the scene (int)

Remove the scene with a certain index.

GetFPS() -> float

No parameters needed

Return the game current FPS.
